# Pyrobot

### `yarn start`

- Launches Node.js server from the source folder (`src/index.js`)

### `yarn run build`

- Builds server into `dist` folder

### `yarn run copy`

- Copies static `.ejs` files from `views` + `package.json` to `dist\views` folder

### `yarn run serve`

- Starts node-js server in `dist` folder
